# java-cli-maven-spring-failsafe-findbugs-junit-test-car-data

## Description
Analyze source code for potential bugs.
A POC for sprimg app using JUnit
framework with failsafe plugin.

## Tech stack
- java
- maven
	- findbugs
  - spring
  - junit
  - failsafe

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
- findbugs report at bin/target/findbugs

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
